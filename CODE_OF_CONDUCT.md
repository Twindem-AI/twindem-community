# Code of Conduct

Twindem uses the Contributor Covenant Code of Conduct, version 2.1.

Project maintainers are responsible for clarifying standards of acceptable
behavior and are expected to take appropriate and fair corrective action in
response to unacceptable behavior.

Examples of behavior that contributes to a positive environment:

- using welcoming and inclusive language,
- respecting differing viewpoints and experiences,
- accepting constructive feedback,
- focusing on what is best for the community.

Examples of unacceptable behavior:

- harassment, insults, or discriminatory language,
- publishing others' private information,
- sustained disruption of discussion or development work,
- any conduct that would reasonably be considered inappropriate in a
  professional setting.

Before public launch, replace this short form with the full Contributor
Covenant text and maintainer contact details.
